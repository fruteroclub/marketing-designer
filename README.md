## Creative Principles

The skill follows **The Olson Method**:

1. Start with emotional truth, not features
2. Reject imitation â€” if it could belong to any brand, it belongs to no brand
3. Separate idea from execution
4. Be willing to start over
5. Empathy is strategic
6. Change the conversation
